# Jeongri

Jeongri is a simple Python-based file organizer designed as a CLI tool. It helps you automatically organize and sort files in your directories based on specific criteria. Right now, the tool focuses on basic file sorting functionality, and it is open to contributions for further development.

---

## Project Overview

Jeongri is a command-line tool that helps you sort files in your directories. It's still in the early stages, with just the file sorting feature implemented. As development continues, additional features and functionalities will be added, but for now, it serves as a basic file organizer for simplifying file management.

The project is open-source and welcomes contributions to help expand its features and functionality.

---

## Current Functionality

- File Sorting: Sort files in a directory based on file types (e.g., images, documents, etc.).

> **Note**: As the project is still in development, features will be expanded upon in future releases.
